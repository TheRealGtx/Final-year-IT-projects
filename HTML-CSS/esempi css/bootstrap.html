Bootstrap

Bootstrap è un framework che ci permette di usare classi html predefinite che imposteranno per noi il layout della pagina rendendolo responsive. Per iniziare a usare bootstrap basterà aggiungere nell'head e nel body del nostro sito i link ai file bootstrap css e js. Una funzione molto utile di bootstrap è la creazione di griglie per posizionare gli elementi. Per farlo si usano dei contenitori div e si inserisce "container", "row" o "col" nel nome della classe. Inoltre è possibile specificare la grandezza delle righe o colonne inserendo sempre nel nome della classe degli attributi appositi. Creiamo una griglia come questa:

1
2
3
4

Scriveremo:
<div class="container">
 <div class="row">
  <div class="col">1</div>
  <div class="col">2</div>
 </div>
 <div class="row">
  <div class="col">3</div>
  <div class="col">4</div>
 </div>
</div>
La maggior parte degli attributi del css sono presenti in bootstrap anche se sono limitati come ad esempio la classe "bg" che permette di impostare un colore di sfondo scegliendo però tra una decina di colori disponibili. Altri attributi di uso comune presenti in bootstrap sono: "m" e "p" rispettivamente per margine e padding, "btn" per i bottoni e "text" per impostare le proprietà del testo. Infine bootstrap permette di utilizzare delle icone che mette a disposizione sul suo sito. Per usare un'icona bisogna inserire nell'head della nostra pagina il link al file delle icone di bootstrap. Per mostrare un'icona la inseriremo in un tag <i>. Ecco un esempio: e il suo rispettivo codice: <i class="bi bi-cake-fill"></i>. Tutte le icone sono visualizzabili dal sito: Icone bootstrap
