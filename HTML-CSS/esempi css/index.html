Il css definisce una serie di regole che consentono di modificare l'aspetto grafico degli elementi
in una pagina HTML. Le regole hanno la sintassi:
Selettore {
proprietà: valore1;
}
Ad esempio ho un tag <div> che voglio di sfondo nero. Il CSS sarà:
div {
background-color: "black";
}

div

Per scrivere il css si hanno tre scelte:

  1. L'attributo style interno a un tag: <div style="background-color: red">
  2. I tag <style> solitamente nella sezione <head> del sito: <style>codice css/>
  3. Fare riferimanto a un file CSS esterno usando <link rel="stylesheet" href="file.css"> nella sezione <head> del sito

Per fare riferimento agli elementi HTML si hanno vari modi detti selettori:

  • Fare riferimento a tutti i tag di quel tipo:
    div {
    background-color: "black";
    }
  • Fare riferimento al nome della classe:
    Ho un tag <div class="quadrato"> il css sarà
    .quadrato {
    background-color: "green"
    }
  • Fare riferimento all'id dell'elemento:
    Ho un tag <div id="altroquadrato"> il css sarà
    #altroquadrato {
    background-color: "purple"
    }

Vedete che abbiamo 3 quadrati ognuno con colori diversi nonostante i tag <div> dovrebbero essere tutti neri.
Questo perchè a ognuno abbiamo dato un colore diverso usando l'id o il nome della classe

Alcuni selettori si attivano solo in corrispondenza di determinati eventi o valori assunti dai tag:

  • Ho un tag <a> con un link e voglio che sia verde se il sito di riferimento è google e rosso se è youtube, scriverò
    a[href$=google.com] {
    background-color: "green";
    }
    a[href$=youtube.com] {
    background-color: "red";
    }
    Avremo: IndovinaIndovina

    $= specifica come deve finire la stringa

  • Ho un bottone e voglio che il suo sfondo cambi quando ci passo sopra con il mouse. Scriverò:
    #bottone:hover {
    background-color: "aquamarine";
    }
  • Ho una checkbox e voglio che diventi rossa quando seleziono l'opzione sbagliata. Scriverò:
    input[type=checkbox]:checked {
    accent-color: "red";
    }
    Non sono bravissimo a presentare

Per applicare le stesse regole a più selettori basterà scrivere i selettori separati da una virgola nel CSS. Se ad esempio ho:

<div id="quadratorosso">rosso</div>
<button id="bottonerosso">Bottone</button>

E li voglio entrambi rossi, nel CSS scriverò:

#quadratorosso, #bottonerosso {
background-color: red;
}


Vediamo come selezionare un selettore contenuto in un altro, come ad esempio un testo in un div:

<div id="quadratonero"><span id="testointerno">Bianco</span></div>

Nel CSS sarà sufficiente scrivere:

#quadratonero > #testointerno {
color: green;
}

Verde

Da notare come non ho specificato il colore del div in quanto a inizio spiegazione abbiamo già definito la regola che impone a tutti i div di essere a sfondo nero.
